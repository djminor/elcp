    First, this website will contain the dictionaries of a host of endangered languages and their translation into English


    Second, this website will translate endangered languages into English


    Third, the translator will be incrementally updated so that it will translate endangered languages into your native language


    Fourth, this website will serve as a hub to learn those endangered languages yourself


    Fifth, the app will be developed and rolled out so that you can learn these languages more conveniently!
